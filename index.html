Dicas para Criar Animações com CSS, HTML e JS

1️⃣ Transições CSS Simples

Use transições para adicionar efeitos suaves às suas animações. Exemplo:

.elemento { transition: all 0.3s ease; }

Ao mudar a propriedade do elemento, ele será animado de forma suave. Experimente passar o mouse sobre o quadrado abaixo:

2️⃣ Animações com @keyframes

Para animações mais complexas, utilize @keyframes. Exemplo:

@keyframes fadeIn {
  0% { opacity: 0; }
  100% { opacity: 1; }
}

.animado {
  animation: fadeIn 1s forwards;
}

3️⃣ Animações com Hover

Adicione interatividade com animações de hover. Exemplo:

.hover-animado {
  transition: transform 0.5s ease;
}

.hover-animado:hover {
  transform: rotate(45deg) scale(1.2);
}

4️⃣ Animação Complexa com @keyframes

Crie animações mais elaboradas usando @keyframes. Exemplo:

@keyframes moveAndFade {
  0% { transform: translateX(0); opacity: 0; }
  50% { transform: translateX(100px); opacity: 1; }
  100% { transform: translateX(0); opacity: 0; }
}

.complexo {
  animation: moveAndFade 3s infinite;
}
